## Sobre o Projeto

Este é um projeto de API REST para gerenciamento de notas, construído com Laravel. O projeto inclui:

- API REST com controladores e recursos
- Sistema de validação de requisições
- Modelo de dados User
- Migrations e factories para testes
- Arquitetura limpa e organizada

## Tecnologias

- Laravel Framework
- PHP
- MySQL
- Git
